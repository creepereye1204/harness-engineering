# docs/PRODUCT_SENSE.md — 제품 방향성

## 이 서비스가 추구하는 것

### 핵심 가치
- **자유로운 소통**: 누구나 게시판을 만들고 커뮤니티를 형성할 수 있다
- **콘텐츠 중심**: 글과 대화가 중심이고 UI는 이를 돕는 도구다
- **건전한 문화**: 신고/제재 시스템으로 최소한의 질서를 유지한다

### 타겟 사용자
- 특정 주제에 관심 있는 사람들이 모여 소통하고 싶은 사람
- 커뮤니티를 직접 만들고 운영하고 싶은 사람
- 익명성보다 닉네임 기반의 반익명 소통을 선호하는 사람

## 기능 우선순위 원칙
새 기능 추가 시 아래 순서로 판단:

1. **핵심 플로우를 방해하는가?**
   - 글쓰기 → 읽기 → 댓글 → 추천 플로우가 최우선
   - 이를 느리게 하거나 복잡하게 만드는 기능은 후순위

2. **사용자가 실제로 원하는가?**
   - 있으면 좋은 기능보다 없으면 불편한 기능 우선

3. **운영 부담을 줄이는가?**
   - 자동 블라인드, 신고 시스템 등 운영 효율 기능 중요
