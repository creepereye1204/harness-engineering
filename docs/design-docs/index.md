# design-docs/index.md — 디자인 시스템

DC인사이드 스타일의 활기차고 컬러풀한 커뮤니티 UI. 라이트/다크 모드 모두 지원.

UI 작업 전 이 파일을 먼저 확인하고, 상세 내용은 각 하위 파일을 참조한다.

## 핵심 원칙
- **콘텐츠 중심** — UI는 글과 댓글을 방해하지 않는다
- **빠른 피드백** — 모든 액션은 즉시 시각적 반응
- **모바일 우선** — 375px 기준으로 먼저 설계, 데스크톱으로 확장
- **일관성** — 새 컴포넌트 추가 전 `components.md`에서 재사용 가능한 것 먼저 확인

## 빠른 참조

### 브랜드 컬러
| 역할 | 값 | 사용처 |
|------|-----|--------|
| Primary | `#FF6B35` | CTA 버튼, 강조 요소 |
| Secondary | `#4ECDC4` | 보조 액션, 뱃지 |
| Accent | `#FFE66D` | 하이라이트, 알림 |

### 레이아웃
| 요소 | 값 |
|------|-----|
| Header 높이 | 56px (sticky) |
| Sidebar 너비 | 240px (1024px 이상 데스크톱만) |
| 콘텐츠 최대 너비 | 1200px |
| 본문 최대 너비 | 800px |
| 브레이크포인트 | 768px / 1024px |

### 폰트
| 용도 | 폰트 |
|------|------|
| 한글 | Noto Sans KR |
| 영문 | Inter |
| 코드 | JetBrains Mono |

### 기술 스택
- CSS: styled-components + ThemeProvider
- 하드코딩 금지 — 반드시 `theme.*` 토큰 사용
- 인라인 스타일 금지
