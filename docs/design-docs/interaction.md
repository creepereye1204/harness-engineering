# design-docs/interaction.md — 인터랙션 규칙

## 애니메이션 원칙
- duration: 150ms (빠른 반응) ~ 300ms (페이지 전환)
- easing: ease-in-out 기본
- 과도한 애니메이션 금지 (콘텐츠 집중 방해)

## 호버 효과
- 버튼: 밝기 10% 감소
- 게시글 카드: 배경색 subtle 변화 + 커서 pointer
- 링크: 언더라인 표시

## 클릭 피드백
- 버튼 클릭: scale(0.97) 100ms
- 추천 버튼: 클릭 시 숫자 +1 즉시 반영 (낙관적 업데이트)
- 비추천 버튼: 동일

## 모달
- 배경 dimmed 처리 (rgba 0.5)
- 배경 클릭 시 닫힘
- ESC 키로 닫힘
- 열기/닫기 fade 애니메이션 200ms

## 무한 스크롤 vs 페이지네이션
- 게시글 목록: 페이지네이션
- 댓글 목록: 더보기 버튼 방식
- 알림 목록: 더보기 버튼 방식

## 다크 모드 전환
- 헤더 토글 버튼으로 전환
- localStorage에 설정 저장
- 전환 애니메이션: 200ms transition
- 시스템 설정 자동 감지 (prefers-color-scheme)
