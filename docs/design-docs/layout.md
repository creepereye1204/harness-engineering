# design-docs/layout.md — 레이아웃 구조

## 전체 구조
```
[Header] - 고정 (sticky), 높이 56px
[Content Area]
  ├── [Sidebar] - 240px 고정 (desktop만)
  └── [Main] - 나머지 전체
[Footer]
```

## Header 구성
- 좌: 로고 + 서비스명
- 중: 검색창
- 우: 알림 아이콘 / 로그인 버튼 (or 프로필)
- 모바일: 검색창 숨김, 아이콘만 표시

## Sidebar 구성 (desktop)
- 구독 중인 게시판 목록
- 인기 게시판 목록
- 게시판 개설 버튼
- 768px 이하에서 숨김

## Main 영역 최대 너비
- 콘텐츠 최대 너비: 1200px (중앙 정렬)
- 게시글 상세 본문 최대 너비: 800px

## 페이지별 레이아웃

| 페이지 | 레이아웃 |
|--------|----------|
| 메인 피드 | Sidebar + 게시글 목록 |
| 게시판 | Sidebar + 게시글 목록 |
| 게시글 상세 | Sidebar + 본문 + 댓글 |
| 글쓰기 | 풀스크린 에디터 |
| 관리자 | Sidebar(관리메뉴) + 콘텐츠 |
| 모바일 전체 | 단일 컬럼 |
