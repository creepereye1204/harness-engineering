# design-docs/typography.md — 타이포그래피

## 폰트
- 한글: Noto Sans KR
- 영문/숫자: Inter
- 코드: JetBrains Mono

## 텍스트 크기 체계
| 이름 | 크기 | 용도 |
|------|------|------|
| xs | 12px | 날짜, 부가 정보 |
| sm | 14px | 댓글, 보조 텍스트 |
| md | 16px | 본문 기본 |
| lg | 18px | 게시글 제목 (목록) |
| xl | 22px | 게시글 제목 (상세) |
| 2xl | 28px | 페이지 제목 |

## 폰트 굵기
- Regular: 400 (본문)
- Medium: 500 (강조)
- Bold: 700 (제목, 버튼)

## 줄간격
- 본문: 1.7
- 제목: 1.3
- UI 요소: 1.0

## 규칙
- 게시글 본문은 md(16px) / Regular / 줄간격 1.7 고정
- 제목은 절대 3줄 이상 넘기지 않음 (말줄임 처리)
- 다크 모드에서 본문 폰트 굵기 한 단계 올림 (가독성)
