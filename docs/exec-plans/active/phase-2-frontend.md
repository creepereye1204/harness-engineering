# docs/exec-plans/active/phase-2-frontend.md — 프론트엔드 구축

## 목표
백엔드 API 완성 후 React 프론트엔드를 순서대로 구현한다.
각 Phase 완료 기준: 실제 API 연동 후 주요 플로우 정상 동작.

---

## Phase 1 — 프론트엔드 기반 세팅
**선행 조건**: 백엔드 Phase 1 완료 (`/actuator/health` 200 응답)

### 작업 목록
- [ ] React 18 + Vite 프로젝트 초기화
- [ ] styled-components + ThemeProvider 설정 (라이트/다크 테마)
- [ ] 라이트/다크 모드 토글 구현 (localStorage 저장, prefers-color-scheme 감지)
- [ ] React Router v6 설정 (라우팅 구조)
- [ ] Axios 인스턴스 설정 (baseURL, 토큰 인터셉터, 401 자동 재발급)
- [ ] React Query 설정
- [ ] Zustand 전역 상태 설정 (인증, 테마, 알림)
- [ ] 공통 컴포넌트 구현 (Button, Input, Toast, Modal, Skeleton, Avatar, Badge, Dropdown, Pagination)

### 완료 기준
- `npm run dev` 실행 후 빈 화면 정상 렌더링
- 라이트/다크 모드 전환 동작 확인
- Axios 인터셉터 단위 테스트 통과 (401 → 재발급 → 재시도 플로우)

## Phase 3 — 게시판 + 게시글 화면
**선행 조건**: 백엔드 Phase 3, 4 완료

### 작업 목록
- [ ] 메인 피드 페이지 (비로그인: 전체 인기글, 로그인: 구독 게시판)
- [ ] 게시판 목록 페이지
- [ ] 게시판 상세 (게시글 목록) 페이지
- [ ] 게시판 개설 페이지
- [ ] 게시글 상세 페이지
- [ ] 리치 에디터 글쓰기 페이지 (TipTap 연동)
- [ ] 이미지 업로드 + 인라인 삽입
- [ ] 임시저장 기능 (자동저장 30초)
- [ ] 게시글 수정/삭제
- [ ] 인기글 탭 (24시간/1주일/전체)
- [ ] 검색 페이지

### 완료 기준
- 게시글 작성 → 목록 확인 → 상세 조회 → 수정 → 삭제 플로우 정상 동작
- 비로그인 상태에서 게시글 읽기 가능, 작성 시도 시 로그인 페이지로 이동 확인
- 임시저장 30초 자동 저장 및 불러오기 확인
- 모바일(375px) 레이아웃 깨짐 없음 확인

## Phase 4 — 댓글 + 추천 화면
**선행 조건**: 백엔드 Phase 5 완료

### 작업 목록
- [ ] 댓글 목록 컴포넌트 (트리 구조, `?lastId=&size=20` 더보기)
- [ ] 댓글 작성/수정/삭제
- [ ] 대댓글 작성/수정/삭제
- [ ] 삭제된 부모 댓글 → 답글 버튼 비활성화
- [ ] 추천/비추천 버튼 (낙관적 업데이트, 실패 시 롤백)
- [ ] 블라인드 게시글/댓글 처리 UI (펼치기 가능)

### 완료 기준
- 댓글 → 대댓글 작성 → 수정 → 삭제 플로우 정상 동작
- 추천 클릭 시 즉시 UI 반영 후 API 호출, 실패 시 롤백 확인
- 삭제된 댓글에 답글 버튼이 노출되지 않음 확인
- 블라인드 컨텐츠 "펼쳐보기" 동작 확인
