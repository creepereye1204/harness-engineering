# Product Specs Index

## 서비스 개요
Reddit/DC인사이드 스타일의 일반 커뮤니티 게시판.
누구나 가입하고, 카테고리별 게시판에 글을 쓰고, 소통할 수 있는 서비스.

## 기능 목록

> ✅ = 명세 작성 완료 (구현 미착수). 구현 진행 상황은 `docs/exec-plans/active/`에서 확인.

| 기능 | 명세 파일 | 명세 상태 |
|------|-----------|-----------|
| 회원가입/로그인/프로필 | user.md | ✅ |
| 게시글 CRUD + 목록/검색 | post.md | ✅ |
| 댓글 + 대댓글 | comment.md | ✅ |
| 추천/비추천 | vote.md | ✅ |
| 게시판 카테고리 분류 | board.md | ✅ |
| 신고 기능 | report.md | ✅ |
| 관리자 페이지 | admin.md | ✅ |
| 알림 기능 | notification.md | ✅ |
| 파일/이미지 첨부 | attachment.md | ✅ |

## 사용자 역할
- **비회원**: 게시글/댓글 읽기만 가능
- **일반회원**: 글쓰기, 댓글, 추천, 신고 가능
- **관리자**: 모든 기능 + 관리자 페이지 접근

## 신규 회원 온보딩 플로우
별도 온보딩 페이지 없음. 아래 순서로 자연스럽게 진입:
1. 회원가입 → 이메일 인증
2. 로그인 후 메인 피드 (전체 인기글)
3. 관심 게시판 구독
4. 구독 게시판 피드로 전환
