# post.md — 게시글 CRUD + 목록/검색

## 기능 설명
게시판에 글을 작성, 조회, 수정, 삭제하는 핵심 기능.
리치 에디터(TipTap) 기반으로 이미지 인라인 삽입, 유튜브 임베드 지원.

## 사용자 시나리오

### 게시글 작성
1. 게시판 메인에서 글쓰기 버튼 클릭 (게시판이 먼저 선택된 상태)
2. 제목 입력
3. 리치 에디터로 본문 작성
4. 임시저장 또는 발행
5. 발행 완료 → 게시글 상세 페이지로 이동

※ 글쓰기는 반드시 특정 게시판 진입 후에만 가능.
  게시판 미선택 상태의 글쓰기는 지원하지 않음.
  post_drafts.board_id는 NOT NULL (db-schema.md 참조)

### 게시글 조회
1. 게시글 클릭 → 상세 페이지 이동
2. 조회수 처리:
   - 로그인: userId 기준 1시간 내 중복 카운트 X
   - 비로그인: IP 기준 1시간 내 중복 카운트 X
   - post_view_logs 테이블로 DB 관리 (초기 구현)
   - 추후 Redis로 전환 예정 (tech-debt-tracker.md 참조)
3. 본문, 댓글, 추천수 표시

### 게시글 수정
1. 작성자 본인만 수정 가능
2. 수정 시 "수정됨" 표시 및 수정 시각 기록
3. 수정 시 기존 첨부 파일 유지 또는 개별 삭제 가능

### 게시글 삭제
1. 작성자 본인 또는 게시판 운영자/관리자 삭제 가능
2. soft delete 처리 (deleted_at 기록)
3. 연관 첨부 파일 전체 soft delete 동시 처리
4. 삭제된 글은 "삭제된 게시글입니다" 표시

### 목록 조회
1. 게시판별 게시글 목록 페이지네이션
   - 파라미터: ?page=0&size=20&sort=createdAt,desc
2. 정렬 옵션: 최신순 / 인기순
3. 인기글 탭 별도 제공

### 검색
1. 제목 검색 / 제목+본문 검색 선택 가능
2. 특정 게시판 내 검색 또는 전체 검색
3. 검색 결과 최신순 정렬

### 임시저장
1. 작성 중 임시저장 버튼 클릭 또는 자동저장 (30초 간격)
2. 임시저장 목록에서 불러오기 가능
3. 임시저장은 최대 10개까지 보관

## 상세 요구사항

### 게시글
- 제목: 1~100자
- 본문: 최소 1자, 최대 50,000자 (HTML 기준)
- 에디터: TipTap
- 저장 형식: HTML (XSS 방지 서버사이드 sanitize 필수)

### 인기글 점수
- 점수 = 조회수 × 0.3 + 추천수 × 0.7
- 점수 갱신 시점:
  - 추천/비추천 발생 시 즉시 갱신
  - 조회수는 성능 이슈로 배치로 주기적 갱신 (10분 간격)
- 최근 24시간 / 1주일 / 전체 기간 탭 제공

### 고정글 (공지)
- 게시판 운영자/관리자만 고정 가능
- 목록 최상단 노출, 최대 3개

## API 목록

| 메서드 | 엔드포인트 | 설명 | 인증 |
|--------|-----------|------|------|
| GET | /api/v1/boards/{slug}/posts | 게시글 목록 | 불필요 |
| POST | /api/v1/boards/{slug}/posts | 게시글 작성 | 필요 |
| GET | /api/v1/boards/{slug}/posts/{postId} | 게시글 상세 | 불필요 |
| PATCH | /api/v1/boards/{slug}/posts/{postId} | 게시글 수정 | 필요 |
| DELETE | /api/v1/boards/{slug}/posts/{postId} | 게시글 삭제 | 필요 |
| GET | /api/v1/posts/popular | 전체 인기글 목록 | 불필요 |
| GET | /api/v1/posts/search | 전체 검색 | 불필요 |
| GET | /api/v1/boards/{slug}/posts/popular | 게시판별 인기글 | 불필요 |
| GET | /api/v1/boards/{slug}/posts/search | 게시판 내 검색 | 불필요 |
| POST | /api/v1/posts/drafts | 임시저장 | 필요 |
| GET | /api/v1/posts/drafts | 임시저장 목록 | 필요 |
| DELETE | /api/v1/posts/drafts/{draftId} | 임시저장 삭제 | 필요 |

## 예외 처리

| 상황 | HTTP 상태 | 에러 코드 | 메시지 |
|------|-----------|-----------|--------|
| 존재하지 않는 게시글 | 404 | POST_NOT_FOUND | "존재하지 않는 게시글입니다" |
| 삭제된 게시글 | 410 | POST_DELETED | "삭제된 게시글입니다" |
| 수정 권한 없음 | 403 | POST_FORBIDDEN | "게시글 작성자만 수정할 수 있습니다" |
| 삭제 권한 없음 | 403 | POST_DELETE_FORBIDDEN | "삭제 권한이 없습니다" |
| 임시저장 한도 초과 | 400 | POST_DRAFT_LIMIT | "임시저장은 최대 10개까지 가능합니다" |
| XSS 감지 | 400 | POST_INVALID_CONTENT | "허용되지 않는 콘텐츠가 포함되어 있습니다" |
